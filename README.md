# IntelligenceOS

Personalized Learning Intelligence Platform:  starting with UPSC.

Built by Amaan Khan
B.Tech CSE, Amity University Noida

## What it does

- Maps daily news to exam syllabus using AI
- Generates PYQ-pattern questions from current affairs
- Evaluates answers and tracks weak topics
- Recommends the next best study action
